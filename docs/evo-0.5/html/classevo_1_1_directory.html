Detailed Description

Directory reader.

  • Use this open a directory and step through the files
Example
#include <evo/dir.h>
#include <evo/io.h>
using namespace evo;
int main() {
SubString filename;
try {
Directory dir(".");
while (dir.read(filename))
con().out << filename << NL;
} EVO_CATCH(return 1);
return 0;
}
