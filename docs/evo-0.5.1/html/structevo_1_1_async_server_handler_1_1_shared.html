Detailed Description

Default shared data (empty) using default global data type.

  • The user defined ServerHandler may define a Shared type that inherits this
  • Caution: Only use this if your ServerHandler does not define a Global type (or use SimpleSharedBase), otherwise you'll get compiler errors
