NewlineDefault Struct Reference

#include <evo/impl/sys.h>

Detailed Description

Default newline type, implicitly converts to NL_SYS (system default newline).

  • This is used to define the NL (default newline) constant, which implicitly converts to NL_SYS (system default newline) with functions taking a Newline type
  • This is useful for having a class differentiate between NL (default newline defined by class) and explicit Newline values like NL_SYS (system default) or nLF
    • Stream does this, ex: an instance of File or Socket can define it's own default newline type used when NL is passed as Newline
    • Passing NL_SYS explicitly uses the system default newline type rather than the class default, or passing nLF explicitly uses that newline value
  • See also: NewlineValue
