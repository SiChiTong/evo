Detailed Description

template<class T>
class evo::AtomicPtr< T >

Atomic dumb pointer to single object.

  • This is an atomic version of Ptr with increment/decrement operator support
  • All operations are atomic
    • Caution: This does not make the pointed type thread safe!
    • Caution: T should be atomic or thread safe, unless you're sure only one thread will use an item at a time
  • Specialized for array pointer: AtomicPtr<T[]>
